# Image Assets For 3D Print Cost Studio

Put your own brand logos and printer photos here. The app will use these local files first.

## Brand Logos

Folder:

`assets/brands/`

Filename:

`brand-name.jpg`

Examples:

`assets/brands/elegoo.jpg`
`assets/brands/bambu-lab.jpg`
`assets/brands/creality.jpg`
`assets/brands/anycubic.jpg`

Recommended size:

1200 x 400 px JPG, white or solid background is fine.
Keep each file under 500 KB if you can.

PNG is still supported as a fallback, but JPG is the preferred format.

## Printer Photos

Folder:

`assets/printers/brand-name/`

Filename:

`model-name.jpg`

Examples:

`assets/printers/elegoo/orangestorm-giga.jpg`
`assets/printers/elegoo/neptune-4-pro.jpg`
`assets/printers/bambu-lab/a1-mini.jpg`
`assets/printers/creality/ender-3-v3-ke.jpg`
`assets/printers/anycubic/kobra-3-combo.jpg`

Recommended size:

1200 x 900 px JPG or 4:3 ratio.
Keep each file under 1 MB if you can.

Placeholder JPG files have already been created for every printer model in the catalog. You can save your real printer photo over the existing file with the same name.

Full filename list:

`assets/PRINTER_PHOTO_FILES.md`

## Naming Rule

Use lowercase English letters and numbers.
Replace spaces and symbols with hyphens.
Use `plus` for `+`.

Examples:

`Bambu Lab` -> `bambu-lab`
`A1 Mini` -> `a1-mini`
`Neptune 4 Pro` -> `neptune-4-pro`
`OrangeStorm Giga` -> `orangestorm-giga`
`X1 Carbon` -> `x1-carbon`
`Form 3+` -> `form-3-plus`
