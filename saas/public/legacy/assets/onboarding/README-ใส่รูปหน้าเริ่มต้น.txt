แฟ้มสำหรับรูปหน้าเริ่มต้น 3D PrintCost Studio

ตำแหน่งแฟ้ม:
outputs/assets/onboarding

ให้วางรูปทับด้วยชื่อไฟล์เหล่านี้:

1. onboarding-logo.png
   - รูปโลโก้/ไอคอนด้านบนกลาง
   - แนะนำขนาด 512 x 512 px
   - ใช้ PNG พื้นหลังใสได้

2. onboarding-fdm.png
   - รูปเครื่องพิมพ์ฝั่ง FDM
   - แนะนำขนาด 1000 x 800 px หรือ 1200 x 900 px
   - ใช้ PNG พื้นหลังใส หรือ JPG พื้นหลังขาวก็ได้

3. onboarding-resin.png
   - รูปเครื่องพิมพ์ฝั่ง Resin
   - แนะนำขนาด 1000 x 800 px หรือ 1200 x 900 px
   - ใช้ PNG พื้นหลังใส หรือ JPG พื้นหลังขาวก็ได้

4. onboarding-background.jpg
   - รูปพื้นหลังจาง ๆ ด้านหลังหน้าต่างเริ่มต้น
   - แนะนำขนาด 1920 x 1080 px
   - ใช้ JPG จะเหมาะที่สุด

5. onboarding-fdm-icon.png
   - รูปไอคอนวงกลมเล็กด้านบนของการ์ด FDM
   - แนะนำขนาด 512 x 512 px
   - ใช้ PNG พื้นหลังใสได้

6. onboarding-resin-icon.png
   - รูปไอคอนวงกลมเล็กด้านบนของการ์ด Resin
   - แนะนำขนาด 512 x 512 px
   - ใช้ PNG พื้นหลังใสได้

หมายเหตุ:
- ถ้ายังไม่มีรูป โปรแกรมจะใช้ภาพเส้นที่อยู่ในตัวโปรแกรมแทน
- ถ้าวางรูปแล้วไม่เปลี่ยน ให้กด F5 เพื่อรีเฟรช
- ชื่อไฟล์ต้องตรงตามนี้ทุกตัวอักษร
